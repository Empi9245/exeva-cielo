# Exeva — Digital Agency

Sito web ufficiale di **Exeva**, agenzia digitale specializzata in web design, branding, marketing e comunicazione.

## Tech Stack

- Next.js
- React
- React Three Fiber
- DREI
- GSAP
- Zustand
- Tailwind CSS

## Configurazione

Per il deploy, imposta le seguenti variabili d'ambiente:

- `NEXT_PUBLIC_SITE_URL` — URL del sito (es. `https://exeva.it`)
- `NEXT_PUBLIC_GA_ID` — ID Google Analytics (opzionale)
- `GH_PAGES_CUSTOM_DOMAIN` — Dominio custom per GitHub Pages (opzionale)

## Sviluppo locale

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
